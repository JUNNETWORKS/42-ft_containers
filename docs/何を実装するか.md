# 実装する必要のあるもの

## 全体

- 基本的にはC++98での各種コンテナのメソッドを実装する
- テストでは出力が正しいかどうかと, 速度をテストする. 速度は遅くてもSTL実装の20倍の遅さまでにする.
- std::allocator を使う必要がある.
- namespaceは `ft::<container_name>` という風にする
